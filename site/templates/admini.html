Gauging Debate: Administrator Console
Here you can start an AWS cluster for tweet tracking on GaugingDebate.com.
If you want to use the default settings (see below), just click "Bake a new cluster".
Otherwise, select "Change Settings", make adjustments, and then click "Bake a new cluster".

You will see regular updates (every 30s) in the red box, informing you of the cluster status.
When the cluster is up and running, tweets will display in the blue box under 'Get tweets'.
At that point, return to the main page to interact with the streaming output.
